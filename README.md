# Nova AI
Nova AI is a Discord Bot with an AI chat and a Voice AI assistant

# About Nova AI
Nova AI uses 
- discord.js
- discord.js/voice
- prism-media
- fs
- child_process
- node-fetch
- path
- Ollama Models

# Commands
- /help (shows all commands)
- /status (shows status of commands)
- /join (joins the voicechannel, the user is in)
- /ada (same as /join but with ada voice from satisfactory)
- /leave (leaves the voice)
- /ai message (chat with custom ollama ai in discord)
